Projeto Web 1

Este projeto é um sistema de controle de acesso baseado em reconhecimento facial. Utilizando Python, uma webcam captura imagens em tempo real e compara com uma base de dados de rostos cadastrados. Se a identidade for confirmada, o sistema libera o acesso, ativando uma catraca eletrônica. Ideal para segurança em ambientes corporativos ou institucionais, o sistema oferece uma solução eficiente e moderna para controle de entradas e saídas.
